# Setup
Pour pouvoir utiliser les lib bluetooth, `F1>ESP-IDF: SDK Configuration editor (Menuconfig)`
Dans `Component config>Bluetooth`
- Cocher Bluetooth
- Host: `Bluedroid - Dual-mode`
- Controller: `Enabled`
